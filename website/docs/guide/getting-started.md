# 快速开始

用最少步骤让 StrokeMouse 跑起来，并验证第一笔手势。

## 你需要什么

- **macOS 14 Sonoma** 或更高
- 任意鼠标（默认用手势的**右键**触发；可在编辑里改成中键等）
- 愿意授予 **辅助功能（Accessibility）** 权限

## 五步上手

### 1. 启动应用

启动后菜单栏会出现 StrokeMouse 图标。应用以菜单栏常驻为主，可在设置中选择是否隐藏 Dock 图标。

### 2. 授予辅助功能权限

打开 **设置 → 权限**，按提示打开系统「隐私与安全性 → 辅助功能」，勾选 StrokeMouse。

::: tip
未授权时引擎**不会**假装在监听。菜单栏与权限页会给出可观测状态，而不是静默失败。
:::

### 3. 确认手势已启用

在菜单栏中确认手势处于启用 /「恢复手势」状态。若曾暂停，先恢复再试。

### 4. 打开默认手势列表

**设置 → 手势** 可查看开箱自带的默认手势（例如上滑 → Mission Control）。也可新建自己的轨迹与动作。

### 5. 画第一笔

1. 按住该手势的 **触发键**（默认**右键**）
2. 在屏幕上拖出轨迹（例如向上滑）
3. 松开

匹配成功后会执行绑定动作。按住触发键时，屏幕上会有**实时轨迹 HUD** 反馈。

## 短按 vs 手势

| 行为 | 结果 |
|------|------|
| 按下触发键后**几乎不移动**就松开 | 视为短按：回放为正常点击，**右键菜单仍可用** |
| 位移超过「最小滑动距离」后松开 | 进入手势识别：整组 down/drag/up 用于手势，**不会**同时弹出右键菜单 |

左键和未配置为触发键的鼠标按钮始终放行，不受影响。
