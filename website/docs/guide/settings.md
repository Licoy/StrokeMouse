# 设置与菜单栏

## 菜单栏

菜单栏图标提供日常控制入口：

- **启停手势** — 临时暂停 / 恢复引擎
- **打开设置** — 进入设置窗口
- **权限相关提示** — 未授权或异常时可见
- **退出**

应用可配置为登录启动、隐藏 Dock 图标，以菜单栏工具形态常驻。

## 设置窗口结构

| 分区 | 内容 |
|------|------|
| **手势** | 列表：增删改、启用开关、进入编辑器 |
| **通用** | 主题（跟随系统 / 浅色 / 深色）、登录启动、Dock 等 |
| **权限** | 辅助功能 / 自动化状态与跳转系统设置 |
| **关于** | 版本与产品信息 |

## 手势列表

- 查看名称、触发键摘要、动作摘要、启用状态
- 新建手势 / 编辑已有项 / 删除
- 默认手势可改可删；也可随时加回自己的模板

## 手势编辑器

编辑单条 `GestureProfile` 时通常包括：

1. **名称** 与备注
2. **触发键**
3. **轨迹** — 录制 free-path 点列，或基于方向模板
4. **动作** — 见 [动作类型](./actions)
5. **作用域** — 全局或指定 App
6. **启用开关**

录制时按住触发键画线，松手完成；可反复录制直到满意。

## 主题与语言

- 界面主题：跟随系统，或强制浅色 / 深色
- 文案：中英 String Catalog；随系统语言切换

## 首次引导

首次启动可能出现 onboarding 引导，提示授权与基本用法。可随时在权限页与文档中回顾。

## 在 Finder 中显示配置

设置中可打开配置文件所在目录，便于备份。详见 [配置文件](./config-file)。
