# 设置与菜单栏

## 菜单栏

菜单栏图标提供日常控制入口：

- **启停手势** — 临时暂停 / 恢复引擎
- **打开设置** — 进入设置窗口
- **状态色** — 正常；**黄色**表示手势已暂停；**红色**表示缺少辅助功能权限
- **退出**

应用可配置为登录启动、隐藏 Dock 图标、**隐藏菜单栏图标**。隐藏菜单栏后，点击 Dock 中的 StrokeMouse 或再次打开 App 会打开设置；若 Dock 也隐藏，通过再次启动 App 进入设置。通用设置中可恢复菜单栏并**退出应用**。

同时隐藏 Dock 与菜单栏时会二次确认，避免失去所有可见入口。

## 设置窗口结构

| 分区 | 内容 |
|------|------|
| **手势** | 侧栏（全局 / 各 App）+ 列表：搜索 / 筛选 / 多选批量操作、导入导出、编辑器 |
| **通用** | 主题、登录启动、隐藏 Dock / 菜单栏、退出应用等 |
| **权限** | 辅助功能 / 自动化状态、引导授权与系统设置 |
| **关于** | 版本与产品信息 |

## 手势列表

- 左侧栏按 **全局** 与**已关联 App** 分组浏览（类似系统设置中的快捷键作用域）；在某一 App 下**新建**会预填对应作用域
- 查看名称、触发键、动作、**作用域（全局或 App 图标）**、启用状态
- **搜索**名称 / 动作 / 备注；按 **全部 / 已启用 / 已禁用** 筛选；表头排序
- 新建 / 编辑 / 删除；**多选**后批量启用、禁用、删除或导出
- **导入 / 导出** JSON 手势包：导出当前选中项；导入时若内容重复，可跳过或强制导入（强制导入的重复项默认关闭）
- 默认手势可改可删；也可随时加回自己的模板

## 手势编辑器

编辑单条 `GestureProfile` 时通常包括：

1. **名称** 与备注
2. **触发键**
3. **轨迹** — 录制 free-path 点列，或基于方向模板
4. **动作** — 见 [动作类型](./actions)
5. **作用域** — 全局，或从已安装应用中按图标搜索 / 多选 / 浏览 `.app` 添加（底层为 bundle id）
6. **启用开关**

录制时按住触发键画线，松手完成；可反复录制直到满意。

## 主题与语言

- 界面主题：跟随系统，或强制浅色 / 深色
- 文案：中英 String Catalog；随系统语言切换

## 首次引导

首次启动可能出现 onboarding 引导，提示授权与基本用法。可随时在权限页与文档中回顾。

## 备份与分享配置

- **日常**：设置 → 手势 → 导出选中项为 JSON；在另一台机器上导入
- **整库**：设置中可「在 Finder 中显示」配置目录，复制 `gestures.json`。详见 [配置文件](./config-file)
